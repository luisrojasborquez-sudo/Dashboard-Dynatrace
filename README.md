# Dashboard-Dynatrace
Template de configuraciones de dashboards de Dynatrace exportadas en formato JSON, listas para ser importadas vía API o interfaz de usuario. Incluye capturas de pantalla de cada dashboard para una referencia visual rápida. Ideal para el control de versiones y el intercambio de configuraciones.
